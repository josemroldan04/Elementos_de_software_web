a) ¿Por qué los enlaces son verdes y no rojos?

Porque existe una regla CSS más específica que le da prioridad al color verde.

Ejemplo:

a {
   color: red;
}

.links a {
   color: green;
}

El navegador aplica verde porque .links a es más específico que solo a.

b) Hacer que los enlaces sean rojos

cambiar la regla específica
.links a {
   color: red;
}

c) Rehacer el HTML usando <div> en lugar de <ul> y <li>

<div class="links">
   <div><a href="#">Inicio</a></div>
   <div><a href="#">Servicios</a></div>
   <div><a href="#">Contacto</a></div>
</div>

d) Reescribir el CSS usando una clase por selector para que se vea igual

Se puede cambiar por:

HTML
<div class="links">
   <a class="link-item" href="#">Inicio</a>
   <a class="link-item" href="#">Servicios</a>
   <a class="link-item" href="#">Contacto</a>
</div>
CSS
.link-item {
   color: green;
   text-decoration: none;
}
