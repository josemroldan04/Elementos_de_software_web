SOLUCION

a) Servidor DNS

Es un sistema que traduce los nombres de dominio en direcciones IP para que los dispositivos puedan localizar recursos en Internet.

b) IP (Dirección IP)

Es un identificador único asignado a cada dispositivo conectado a una red, que permite su ubicación y comunicación.

c) Protocolo

Es un conjunto de reglas que definen cómo se transmiten y reciben datos entre dispositivos en una red.

d) Nombre de dominio

Es la forma legible y fácil de recordar que identifica a un sitio web en Internet.

e) Directorio y nombre de archivo

Es la estructura que indica la ubicación específica de un recurso dentro de un sitio web, incluyendo carpetas y el archivo solicitado.

Gestiona, organiza y protege los datos del sistema, garantizando su integridad, disponibilidad y buen rendimiento.

Planifica, coordina y supervisa el desarrollo del proyecto, asegurando el cumplimiento de objetivos, tiempos y recursos.
